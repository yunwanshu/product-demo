Prompt 自迭代反馈循环 Agent

Agent 对比新旧版本评分差异，定位问题维度，生成修改建议

美国 · 服饰 · V3 → V4 迭代分析
信息完整性7.8 → 8.2+0.4
可读性7.5 → 7.8+0.3
合规性9.3 → 9.5+0.2
搜索友好度6.8 → 7.1+0.3
Agent 诊断：V3 增加了"trending""viral"等社媒用语禁止列表后，合规性和搜索友好度同步提升。V4 无显著回归维度。
建议：当前版本已达边际收益递减点（V3→V4 综合提升仅 +0.3），建议停止迭代，转向新类目。
德国 · 工具 · V1 → V2 迭代分析
信息完整性7.2 → 6.5-0.7
可读性6.8 → 7.9+1.1
合规性8.5 → 8.8+0.3
搜索友好度6.2 → 6.8+0.6
Agent 诊断：V2 强制使用德语技术术语（Akkuschrauber 替代 cordless drill），可读性大幅提升但信息完整性下降——部分德语术语过于专业，买家搜索时仍用英语借词。
建议：V3 应增加"德语专业词 + 英语借词双语并存"规则，如 "Akkuschrauber (Cordless Drill)"。
